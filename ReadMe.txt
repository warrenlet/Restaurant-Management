How to run the restaurant customer management app:

Step 1: Run the 'restaurantManagement' sql script to create the restaurant database
Step 2: Use any java compiler to open the 'Restaurant_customer_management' folder as a project
Step 3: After the compiler access the project folder, go to the 'sqlConnection.java' and follow the comment to set up database connection
Step 4: Go to the 'MainInterface.java' and run the program
